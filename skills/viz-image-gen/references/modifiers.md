# Prompt Modifiers — Organised by Framework Element

Reliable modifiers that consistently influence output across both GPT and Gemini. Use 2-3 focused modifiers per prompt — more than 5 competing modifiers degrades coherence.

Organised by the 6-Element Framework for easy discovery when building Visual Breakdowns.

---

## 1. Subject Modifiers

| Modifier | Effect | Best for |
|----------|--------|----------|
| `extreme detail` | Every surface rendered with forensic precision | Hyperreal, macro |
| `simple character sketch` | Minimal detail, expressive | Comics, storyboards |
| `environmental storytelling` | Scene tells the story, subject secondary | Cinematic, wide shots |
| `product hero` | Object is sculptural, center of attention | Product photography |
| `authentic imperfection` | Visible pores, freckles, wrinkles, scars | UGC, portraits |
| `dynamic action` | Motion blur, mid-movement capture | Action, sports, energy |
| `silhouette` | Subject as dark shape against light | Cinematic, dramatic |

## Combining Modifiers

**Good combinations (2-3 focused, all reinforcing one direction):**
- "85mm portrait lens, Rembrandt lighting, film grain" → cinematic portrait
- "flat vector illustration, soft pastel palette, clean white background" → brand icon
- "100mm macro, soft diffused light, shallow DOF" → product detail
- "wide-angle, golden hour, volumetric light" → atmospheric landscape
- "anamorphic 2.39:1, neon glow, volumetric light" → cinematic sci-fi
- "smartphone (iPhone), ring light, authentic imperfection" → UGC content
- "medium format, hard side light, wet surface" → luxury product

**Bad combinations (conflicting):**
- "minimalist" + "highly detailed" → contradictory
- "vintage" + "futuristic" → unclear direction
- "photorealistic" + "watercolor" → competing mediums
- "flat light" + "chiaroscuro" → impossible
- 10+ modifiers → model loses coherence

**Principle:** 2-3 focused modifiers beat 10 competing ones. Each modifier should reinforce a single coherent visual direction established by the Visual Breakdown.
